# Security Policy

## Supported versions

Only the latest release is supported with security fixes.

## Reporting a vulnerability

Please do not open a public issue for security problems. Instead, email
**adityaryawork@gmail.com** with a description and, if possible, reproduction
steps.

Expect an initial response within **~72 hours**. We'll work with you to
confirm the issue, land a fix, and coordinate disclosure timing.
